## Installation
**Requirements**
```
python >= 3.8
pandas >= 1.4
numpy >= 1.22
```

<br>

You can install this package with [poetry](https://python-poetry.org/docs/) or [pip](https://pypi.org/project/pip/)
<br>
poetry:
```console
poetry add data-hopper
```
pip:
```console
pip install data-hopper
```
